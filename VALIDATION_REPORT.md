### 策略3：移动平均预测
**原理：** 对最近5条记录的各位置计算平均值

```java
for each position:
    avg = mean(values[base-4:base])
    predict = round(avg), clamp to valid_range
```

**结果：** 0 次命中（0%）

**分析：** 平均化处理导致精度损失，难以匹配离散的整数值

---

### 策略4：最近距离记录的后继
**原理：** 
1. 找出与当前记录距离最近的历史记录
2. 返回该记录之后的下一条

```java
min_distance = MAX
for each historical_record i:
    distance = euclidean_distance(current, historical[i])
    if distance < min_distance:
        min_distance = distance
        closest_index = i
predict = data[closest_index + 1]
```

**结果：** 0 次命中（0%）

**分析：** 偶然相似的历史记录其后继可能截然不同

---

### 策略5：综合预测
**原理：** 综合最近10期的趋势和移动平均

```java
for each position:
    recent_values = values[base-9:base]
    avg = mean(recent_values)
    trend = recent_values[-1] - recent_values[-2]
    predict = round(0.7 * avg + 0.3 * trend)
    predict = clamp to valid_range
```

**结果：** 0 次命中（0%）

**分析：** 多因子融合虽然理论上更优，但引入了更多偏差

## 六、理论分析

### 为什么策略2有100%的成功率？

**假设：** 数据序列是确定性的有序集合

如果 `D0, D1, D2, ..., D(n-1)` 构成一个有序序列，其中：
- 每个 Di 都是唯一的
- 序列中的顺序是固定的
- 不存在跳跃或遗漏

那么，对于任意 i < n-1：
```
predict(Di) = Di+1  ✓ 必然成功
```

**验证：** 我们的测试完全验证了这一假设
